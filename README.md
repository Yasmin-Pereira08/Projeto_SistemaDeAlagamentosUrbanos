# Sistema-de-detec-o-de-Alerta-de-Alagamentos-urbanos
Sistema de detecção de alagamentos desenvolvido no SENAI – Serviço Nacional de Aprendizagem Industrial. Sensores enviam dados via serial para aplicação C#/.NET WPF que classifica o risco, exibe no mapa e recalcula rotas via serviços da Google. Dados persistidos em MySQL.
